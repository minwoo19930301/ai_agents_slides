# AI Agents Slides

AI 에이전트 서비스 방식, 요금제, 웹/API 차이, AFK 에이전트 흐름을 설명하는 HTML 슬라이드 덱입니다.

## Links

- Cloudflare Pages: https://ai-agents-slides.pages.dev
- GitHub repository: https://github.com/minwoo19930301/ai_agents_slides
- GitHub Pages: https://minwoo19930301.github.io/ai_agents_slides/

## Files

- `index.html`: 발표용 HTML 슬라이드
- `slides-data.js`: 슬라이드 로딩 순서
- `slides/`: 슬라이드별 구성 파일
- `assets/`: 슬라이드에 들어가는 이미지와 영상 자료
- `slide-summary.md`: 전체 슬라이드 요약
- `ai_agents_slides_image_pack.pptx`: 주요 이미지 자료를 16:9 PPT 슬라이드로 정리한 파일

## Deployment Note

GitHub Pages가 이 저장소의 `main` 브랜치를 바라보도록 설정되어 있으면, GitHub에 push한 뒤 GitHub Pages는 자동으로 갱신됩니다.

Cloudflare Pages는 현재 직접 업로드 방식으로 배포한 상태라서, GitHub push만으로 자동 갱신되지 않을 수 있습니다. Cloudflare Pages에서도 자동 갱신되게 하려면 GitHub 저장소를 Cloudflare Pages 프로젝트에 연결해야 합니다.
